# 롤로노아


## 소개
롤로노아는 리그오브레전드(https://www.leagueoflegends.com) 게임의 API(https://developer.riotgames.com/apis)를 사용하여 만든 서비스로 본인 게임 플레이의 피드백을 받고싶은 사람, 같이 게임을 한 사람들의 리뷰를 남기고 싶은 사람을 목표로 합니다.


## 기능
- 1 vs 1 관점 싸움 피드백
- 본인 플레이 피드백
- 리뷰 남기기
- 유저 검색
